# JS-Star-Wars-RPG

Instructions:

- The 4 characters are selectable. First character becomes your player. The rest become the enemies.
- One by one defeat the enemies.
- Every character has a health and fight power.
- When you attack you decrease the opponent's health by your attack power and vice versa. Only your attack power increases as the game continues but the opponent's attack power stays the same.


Game Info:

The game is built using Bootstrap, Javascript and JQuery. It relies heavily on dynamic DOM manipulation and the use of on click functions.
